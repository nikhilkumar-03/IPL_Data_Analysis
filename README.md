# IPL Data Analysis

Exploratory Data Analysis on Indian Premier League cricket data
covering seasons 2008 to 2022.

## What This Project Covers
- Team performance and win counts
- Toss impact on match results
- Top run scorers and wicket takers
- Player of the Match trends
- Most used venues

## Tools Used
Python, Pandas, Matplotlib, Seaborn, Jupyter Notebook

## Dataset
IPL Complete Dataset from Kaggle — covers 800+ matches
and 200,000+ ball by ball deliveries.

## How to Run
1. Clone this repository
2. Open IPL_Data_Analysis.ipynb in Jupyter Notebook
3. Run all cells
